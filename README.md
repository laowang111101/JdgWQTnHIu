# 在线自主评测系统 python1160

## 项目概述

在线自主评测系统是一款基于Python开发的在线考试管理系统。系统提供了题库管理、考试组织、成绩管理、用户管理等功能模块，实现了考试的自动化和智能化。

## 技术栈

- **后端**: Python 3.7, Django
- **前端**: Vue
- **数据库**: MySQL
- **开发工具**: PyCharm

## 功能模块

### 用户管理

- 账号注册与登录：学生可使用学号进行注册和登录。

### 考试管理

- 在线考试：学生登录后可参加在线考试。
- 考试时间控制：自动记录考试时间，考试结束自动保存答案。
- 提前交卷：学生可选择提前交卷。

### 成绩管理

- 成绩查询：学生可查看个人考试成绩。
- 成绩排名：展示学生考试成绩排名。
- 成绩统计：后端自动计算和统计考生成绩。

### 题库管理

- 随机组卷：根据题库随机生成试卷。
- 题目与附件管理：自动匹配题目与附件。

## 系统角色

- 学生：参加考试，查询成绩。
- 教师：管理题库，组织考试，查看成绩统计。
